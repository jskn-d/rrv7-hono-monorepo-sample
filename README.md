## Tech Stack

[bun](https://bun.sh/)  
[React Router v7](https://reactrouter.com/en/main) (Framework)  
[Hono](https://hono.dev/)

## Getting Started

Install the dependencies.
```bash
bun install
```

Run the command.
```bash
bun run --filter '*' dev
```
